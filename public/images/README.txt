GIBBS TOWING - IMAGE FILES NEEDED
==================================

Add your truck photos to this folder with these exact names:

REQUIRED:
---------
hero-truck.jpg    - Main hero image (your best truck photo, landscape, high-res)
favicon.png       - Small square logo/icon (at least 180x180px)
owner.jpg         - Photo of James Gibbs or the team

FLEET GALLERY (add as many as you have):
----------------------------------------
fleet-1.jpg       - Heavy Duty Wrecker Fleet
fleet-2.jpg       - Night Operations & Recovery
fleet-3.jpg       - ATL Load Shifts
fleet-4.jpg       - Rotator Service
fleet-5.jpg       - Heavy Transport
fleet-6.jpg       - Professional Recovery

TIPS:
- Use .jpg or .png format
- Landscape orientation works best for hero and fleet images
- Recommended size: 1200px wide minimum for quality
- Keep file sizes under 500KB each for fast loading

After adding images, commit and push to GitHub:
  git add .
  git commit -m "Add truck images"
  git push

Vercel will auto-deploy with your new images!
