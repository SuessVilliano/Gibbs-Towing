# Gibbs Towing Company Images

## Instructions
Place your actual company truck images in this directory:

1. **Hero Background Image**: 
   - Filename: `gibbs-hero-truck.jpg` or `gibbs-hero-truck.png`
   - Recommended size: 1920x1080 or larger
   - This will be the main background image on the homepage

2. **Truck Animation Images**:
   - Filename: `gibbs-truck-1.jpg`, `gibbs-truck-2.jpg`, `gibbs-truck-3.jpg`, etc.
   - Recommended size: 800x600
   - These will scroll across the bottom of the page

3. **Fleet Gallery Images**:
   - Add any additional fleet images here
   - They can be managed through the admin panel at: `yourdomain.com/#admin-fleet-2024`

## Current Setup
The website is currently using placeholder images from Unsplash. Once you add your actual truck images here, the code will automatically use them.

## Image Formats Supported
- JPG/JPEG
- PNG  
- WebP (recommended for best performance)
